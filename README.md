# sticker-tg-bot
Sticker telegram bot

## Steps
1. Clone repo to your computer with git clone {repoURL}
2. Create your working branch with git checkout -b {branchName}
3. When u make any changes on your machine which u want to push remotelly: 
    1. git add {filesToPush}
    2. git commit -m "Message of commit"
    3. git push
